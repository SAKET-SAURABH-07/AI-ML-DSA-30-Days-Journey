# FIFA World Cup 2026 Player Clustering using K-Means

## Project Overview

This project groups FIFA World Cup players into clusters using the K-Means clustering algorithm.

## Features

- Data Preprocessing
- StandardScaler
- K-Means Clustering
- Elbow Method
- Silhouette Score
- PCA Visualization
- Interactive Streamlit Dashboard
- Download Clustered Dataset

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Streamlit

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
